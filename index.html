Meaning of some Terminologies

padding

Padding is used to create space around an element's content,
inside of any defined borders.

Margin

The margin CSS shorthand property sets the margin area on all four sides of an element.

Border

The border property is a shorthand syntax in CSS that
accepts multiple values for drawing a line around the element it is applied to.
